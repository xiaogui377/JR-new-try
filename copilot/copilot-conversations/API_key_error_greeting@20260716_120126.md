### 1. 顶层设计 — 创建“电路”MOC
新建一个笔记，标题为 [[电路 MOC]]，作为本课程的导航中心。它里面不存具体知识，只负责链接各章节、重点笔记和复习计划。

**示例内容结构：**
```
# 电路复习地图

## 章节导航
- [[电路模型与基本变量]]
- [[电阻电路等效变换]]
- [[电路分析方法]]
- [[电路定理]]
- [[动态电路（一阶/二阶）]]
- [[正弦稳态电路]]
- [[耦合电感与变压器]]
- [[三相电路]]
- [[非正弦周期电流电路]]
- [[二端口网络]]

## 复习工具
- [[公式速查表（电路）]]
- [[典型题型汇总]]
- [[易错点清单]]

## 学习日志
- [[2026-07-16 电路复习]]
```

### 2. 章节笔记结构模板
为每一章创建独立笔记，使用统一模板，方便批量生成和后续 Dataview 统计。

**模板示例（可命名为 `电路章节模板`）：**
```
---
tags:
  - 电路/章节
date: {{date}}
进度: "未开始"
---

# {{title}}

## 核心概念
- 

## 重要公式
- 

## 考点分析
- 

## 例题链接
- [[例-XXXX]]
```

用 Templater 插入当前日期和标题，填写一章后就可以通过标签 `#电路/章节` 汇总所有章节的进度。

### 3. 题型与错题笔记
为每道题单独创建笔记，利用双链关联到章节和考点。

**错题/例题模板：**
```
---
tags:
  - 题型/选择
  - 考点/节点电压法
  - 来源/真题2024
掌握程度: "薄弱"
---

# {{title}}

## 题目
（描述或截图）

## 错解/常见误区
- 

## 正确解答
- 

## 关联章节
- [[电路分析方法]]
```

后期用 Dataview 筛选 `掌握程度 = "薄弱"` 的题目，反复攻克。

### 4. 公式与图像管理
- 每个公式在笔记内用 LaTeX 书写，例如 $I = \frac{V}{R}$。
- 电路图建议用 Excalidraw 绘制后嵌入，图片自动存入 `附件` 文件夹，笔记内用 `![[image.png]]` 引用。
- 可以创建一个 [[公式速查表（电路）]] 专门汇总所有公式，如 KCL/KVL 表达式、戴维南参数、一阶响应公式等。

### 5. 标签体系
建议分层标签，方便多维筛选：
- `电路/章节`：标注章节笔记
- `电路/概念`：核心概念
- `题型/计算、题型/选择、题型/填空`：区分题型
- `考点/XXX`：具体考点，如 `考点/戴维南定理`、`考点/三相功率`
- `掌握程度/熟练、掌握程度/一般、掌握程度/薄弱`

### 6. 复习计划驱动
- 可在 [[电路 MOC]] 中嵌入 Dataview 查询，自动列出薄弱环节：
  `\`\`\`dataview TABLE 掌握程度, 关联章节 FROM #考点  WHERE 掌握程度 = "薄弱" SORT 关联章节 ASC \`\`\``
- 在每日笔记中写 `![[电路 MOC]]` 提醒自己复习当天的薄弱章节。
